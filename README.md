# LAST BASTION

A 3D tower defense game with a bioluminescent dark fantasy aesthetic, built with Three.js.

## Play

```bash
npm install
npm run dev
```

## Features

- **4 Tower Types**: Arrow, Frost, Cannon, Laser â€” each with 3 upgrade levels
- **5 Enemy Types**: Grunt, Brute, Swarm, Healer, Boss
- **20+ Waves** of escalating difficulty with boss fights every 5 rounds
- **Bioluminescent aesthetic** with bloom, god rays, particles, and dynamic lighting
- **Economy system** with gold, interest, tower selling, and upgrade paths
- **Full UI/HUD** with wave preview, tower stats, and upgrade panels
- **Synthesized audio** via Web Audio API
- **Graphics quality settings** (Low / Medium / High)

## Controls

- **Left Click**: Select/place towers
- **Right Click + Drag**: Pan camera
- **Scroll Wheel**: Zoom in/out
- **ESC**: Pause
- **1-4**: Quick-select tower types

## Tech Stack

- Three.js + postprocessing
- Vite
- Vanilla JS (no frameworks)
